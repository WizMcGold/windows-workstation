Mac OS X Yosemite cursors for Windows
-------------------------------------

Download
-------
Download cursor files at: http://laccelo.deviantart.com/art/Mac-OS-X-Yosemite-Cursors-for-Windows-XP-7-8-8-1-514893122

Install
-------
- Extract the archive
- Right click the on an "_cursor_ OS X Yosemite.inf" file and select "Install"
- Choose the cursor set in the Control Panel mouse applet

You can open mouse applet by
- pressing Win + R
- typing: main.cpl
- pressing ENTER
